# A-Stock Analysis Report Template (A股分析报告模板)

## Report Structure (报告结构)

### 1. Executive Summary (执行摘要)

简洁的2-3段概述，可以独立阅读：
- **公司概述** (1句话)
- **投资论点** (看多理由，2-3句话)
- **主要风险** (1-2句话)
- **估值摘要** (当前价格vs合理估值)
- **投资建议** (买入/持有/卖出) + 信心水平

### 2. Company Overview (公司概况)

- **公司简介**: 主营业务、主要产品/服务
- **所属行业**: 行业分类和行业动态
- **市场地位**: 市值、行业排名
- **关键统计数据**: 
  - 股票代码、交易所
  - 总市值
  - 当前股价
  - 52周区间
  - 平均成交量

### 3. Investment Thesis (投资论点)

**看多理由** (Why to invest):
- 3-5个最强的正面因素
- 每个因素用数据/证据支持
- 尽可能量化

**看空理由** (Risks and concerns):
- 3-5个主要风险因素
- 投资不成立的替代情景
- 可能时进行概率评估

### 4. Fundamental Analysis (基本面分析)

**业务质量**
- 竞争优势/护城河
- 管理层质量评估
- 业务模式可持续性

**财务表现** (使用表格)
- 收入趋势 (3-5年)
- 盈利能力指标趋势
- 资产负债表强度
- 现金流分析
- 关键比率

**成长性分析**
- 历史增长率
- 增长驱动因素
- 未来增长前景
- 市场机会规模

### 5. Valuation Analysis (估值分析)

**当前估值指标** (表格格式)
- 市盈率 (当前、前瞻、5年平均)
- PEG
- 市净率
- 市销率
- EV/EBITDA
- EV/Sales
- 股息率 (如适用)

**同业比较** (表格格式)
- 与3-5个直接竞争对手的关键指标比较

**合理估值估计**
- 使用的估值方法
- 关键假设
- 合理估值区间 (保守到乐观)
- 相对于当前价格的安全边际
- 目标价格及时间框架

### 6. Technical Analysis (技术分析)

**趋势分析**
- 当前趋势方向
- 关键支撑/阻力位
- 移动平均线位置

**指标摘要**
- RSI读数和解释
- MACD状态
- 成交量趋势
- 其他相关指标

**图表形态**
- 正在形成或已完成的重要形态
- 需关注的突破/跌破水平

**技术面展望**
- 短期 (1-3个月)
- 中期 (3-6个月)

### 7. Risk Assessment (风险评估)

**公司特定风险** (按优先级)
1. [最高风险及影响和概率]
2. [中等风险]
3. [较低风险]

**行业风险**
- 监管风险
- 竞争风险
- 技术颠覆风险

**宏观风险**
- 经济周期风险
- 利率风险
- 汇率风险 (如适用)
- 地缘政治风险

**风险评级**: 高/中/低

### 8. Investment Recommendation (投资建议)

**总体评级**: [强烈买入/买入/持有/卖出/强烈卖出]

**评级理由**: 
- 2-3句话总结核心投资逻辑

**目标价格**:
- 12个月目标价: ¥XXX
- 上涨空间: XX%

**建议仓位**: 
- 激进投资者: X%
- 稳健投资者: X%
- 保守投资者: X%

**关键催化剂**:
- [即将发生的重要事件]
- [需要关注的关键指标]

**止损建议**:
- 技术止损位: ¥XXX
- 基本面止损条件: [XXX]

### 9. Appendix (附录)

**数据来源**: 
- AKShare
- 东方财富
- 腾讯财经
- [其他数据源]

**免责声明**: 
本报告仅供参考，不构成投资建议。投资者应根据自身情况独立决策，风险自担。

## Formatting Guidelines (格式指南)

### Tables (表格)
- 使用Markdown表格格式
- 对齐数字列（右对齐）
- 包含适当的单位
- 使用千位分隔符（1,000,000）

### Charts (图表)
- 价格走势图
- 估值对比图
- 财务指标趋势图
- 技术指标图

### Color Coding (颜色编码)
- 绿色: 正面/看涨
- 红色: 负面/看跌
- 黄色: 中性/警告

### Emphasis (强调)
- **粗体**: 关键指标和重要发现
- *斜体*: 定义和解释
- `代码`: 股票代码和技术术语

## Quality Checklist (质量检查清单)

在发布报告前，确保:

- [ ] 所有数据都是最新的
- [ ] 计算经过验证
- [ ] 没有打字错误
- [ ] 逻辑一致
- [ ] 投资建议明确
- [ ] 风险被充分披露
- [ ] 格式统一
- [ ] 图表清晰
- [ ] 引用数据来源
- [ ] 包含免责声明
